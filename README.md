This repository contains a scripts for processing viral fraction reads from kraken2 sensitive profiling for analysis "HPV infection restructures the cervicovaginal virome along a bacteriophage–papillomavirus axis linked to bacterial dysbiosis and lesion progression".

```0-process-kraken-data.ipynb``` - This notebook provides a step-by-step explanation of the kraken2 viral fraction processing steps.

```1-process-genomad-pharokka.ipynb``` - This notebook provides a step-by-step explanation of the viral functional GeNomad and Pharokka annotation processing steps.
